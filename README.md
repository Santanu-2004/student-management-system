# 🎓 Student Management System

A full-stack web application to manage student data efficiently. Users can submit student details through a form, store them in a database, and view/manage records via a dashboard.

---

## 🚀 Features

- 📝 Student Form (Name, Degree, Department, Year, Enrollment, Mobile)
- 💾 Data stored in MongoDB
- 📊 Dashboard to view all students
- ❌ Delete student records
- 🔄 Auto form reset after submission
- 🌐 REST API using Express.js

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB (Mongoose)
